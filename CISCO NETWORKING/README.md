## part 01 네트워크 세상에 들어서며

#### 01. 네트워킹 인터넷 인트라넷 엑스트라넷

> 네트워킹, 인터넷, 인트라넷, 엑스트라넷

1. 네트워킹 Networking

- 장비들을 서로 대화가 가능하도록 묶어주는 것

- 서로 연결된 장비들끼리 대화를 주고받을 수 있어야 한다
  - for 정보의 공유, 자원의 공유

2. 인터넷 Internet

- Inter 연결 + Net : "여러 개의 네트워크를 묶었다" , 네트워크를 여러 개 묶어놓은 네트워크 연결
- Web : World Wide Web

3. 인터넷의 특징

- 하나의 프로토콜만 사용 (TCP/IP라는 공통의 프로토콜 사용)
- 주로 익스플로러 크롬 파이어폭스와 같은 웹 브라우저를 이용해서 인터넷을 탐험한다

3. 인트라넷 IntraNet

- Intra 내부의 + Net : "내부의 네트워크"
- 똑같이 TCP/IP 프로토콜 사용, 웹 브라우저 이용해서 마치 인터넷 사용하듯이 사내 업무 처리
- VS 인터넷 : 그 회사 직원 말고는 인터넷을 통해서 접속 불가

4. 엑스트라넷 ExtraNet

- alike 인트라넷 : 사용 범위를 직원 이외에도 협력 회사,고객까지 확대
